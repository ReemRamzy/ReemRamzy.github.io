## About Me

I am a **Senior Full Stack Engineer** experienced in developing scalable web applications, mobile backends, fintech systems, logistics platforms, and real-time products.

My main stack includes **Laravel, PHP 8+, Python, Django, Vue.js, Nuxt.js, React.js, and Next.js**. My work goes beyond implementing isolated features: I design APIs, structure backend services, optimize databases, integrate external platforms, build operational dashboards, deploy applications, investigate production issues, and support reliable delivery from planning through maintenance.

I have also led assigned technical work, reviewed code, supported other developers, estimated tasks, and coordinated delivery with product, frontend, and QA teams.

> **What I focus on:** clean backend architecture, reliable APIs, real-time communication, secure payment flows, maintainable code, and production stability.
